# Техническое задание — Skrinshot

## Проект

Desktop-приложение для скриншотов и записи экрана с загрузкой файлов на локальный сервер в одной сети.

## 1. Общая информация

### 1.1 Назначение проекта

Необходимо разработать desktop-приложение для Windows, предназначенное для:

- создания скриншотов экрана;
- записи видео экрана;
- базового редактирования скриншотов;
- локального сохранения файлов;
- загрузки файлов на сервер в локальной сети;
- получения ссылки или пути к загруженному файлу.

Приложение должно быть по UX и сценарию использования максимально приближено к классическому «Скриншотеру»: быстрый вызов по горячей клавише, захват области, мгновенное сохранение, копирование и отправка.

### 1.2 Основная задача

Создать удобный, быстрый и стабильный инструмент для ежедневной работы внутри локальной сети без зависимости от облачных сервисов.

### 1.3 Целевая платформа

- Windows 10
- Windows 11

### 1.4 Язык интерфейса

- Русский по умолчанию

## 3. Требования к UX/UI

### 3.1 Основной сценарий работы

Приложение работает в фоне в системном трее.

### 3.2 Контекстное меню трея

- Скриншот
- Скриншот области
- Запись экрана
- Запись области
- История
- Настройки
- Выйти

Во время записи меню заменяется на: **Остановить запись** / Выйти.

### 3.3 Горячие клавиши по умолчанию

| Действие | Клавиши |
|---|---|
| Скриншот всего экрана | `PrintScreen` |
| Скриншот области | `Ctrl+PrintScreen` |
| Запись экрана | `Shift+PrintScreen` |
| Запись области | `Ctrl+Shift+PrintScreen` |

## 4. Требования к серверной части

### 4.1 API Endpoints

- `POST /api/upload/image`
- `POST /api/upload/video`
- `GET /api/files/:id`
- `GET /api/health`

### 4.2 Формат запроса

- `multipart/form-data`
- Поле файла: `file`
- Авторизация: `Authorization: Bearer <token>` (опционально)

### 4.3 Пример JSON-ответа

```json
{
  "success": true,
  "fileId": "abc123",
  "fileName": "screenshot_2026-03-15_22-51-14.png",
  "url": "http://192.168.1.100:8080/files/abc123",
  "size": 245821
}
```

## 5. Именование файлов

| Тип | Шаблон по умолчанию |
|---|---|
| Скриншот | `screenshot_{YYYY}-{MM}-{DD}_{HH}-{mm}-{ss}.png` |
| Видео | `video_{YYYY}-{MM}-{DD}_{HH}-{mm}-{ss}.webm` |

Шаблон настраивается. Переменные: `{YYYY}`, `{MM}`, `{DD}`, `{HH}`, `{mm}`, `{ss}`.
