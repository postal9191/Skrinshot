# Skrinshot

Десктопное приложение для Windows: скриншоты, запись экрана, аннотации и загрузка на локальный сервер.

## Возможности

### Скриншоты
- **Весь экран** — мгновенный захват, открывается встроенный редактор
- **Область экрана** — выделение произвольной области, редактор появляется прямо поверх выделения (как в Screenshotter)

### Редактор аннотаций
- Инструменты: стрелка, линия, карандаш, маркер, прямоугольник, эллипс, текст
- Выбор цвета и толщины линии
- Отмена последнего действия (Ctrl+Z)
- Действия: **Сохранить на диск** / **Скопировать в буфер** / **Загрузить на сервер** / **Закрыть**

### Запись экрана
- **Весь экран** — запись полного экрана
- **Область экрана** — выделение области перед записью, запись только выбранного фрагмента
- Индикатор REC внизу экрана, рабочий стол полностью доступен во время записи
- После остановки — превью с выбором: сохранить локально или загрузить на сервер
- Формат: WebM (VP9), до 5 Mbps, настраиваемые FPS: 15 / 30 / 60

### Загрузка на сервер
- HTTP API, `multipart/form-data`, Bearer Token авторизация
- Автоматическая загрузка после создания (опционально)
- Очередь при недоступности сервера, повторная отправка
- Ссылка на файл копируется в буфер обмена

### История
- Список скриншотов и видео с датой, размером, статусом и ссылкой
- Удаление записей из истории

### Настройки
- Папка сохранения, шаблон имени файла
- Формат изображений (PNG / JPG), качество JPG
- Параметры видео (FPS, битрейт)
- Адрес сервера и Bearer Token
- Горячие клавиши — capture-поля (кликните на поле и нажмите комбинацию)
- Автозапуск с Windows

## Горячие клавиши по умолчанию

| Действие | Клавиши |
|---|---|
| Скриншот всего экрана | `PrintScreen` |
| Скриншот области | `Ctrl+PrintScreen` |
| Запись экрана | `Shift+PrintScreen` |
| Запись области | `Ctrl+Shift+PrintScreen` |

Все клавиши настраиваются в разделе Настройки.

## Меню трея

Приложение работает в фоне в системном трее. Правый клик на иконке:

- Скриншот
- Скриншот области
- Запись экрана
- Запись области
- История
- Настройки
- Выйти

Двойной клик — открыть главное окно.

## Установка и запуск

### Зависимости

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

### Сборка установщика

```bash
npm run electron:build
```

Установщик: `dist-build/Skrinshot Setup 1.0.0.exe`

## Структура проекта

```
src/
  main/
    main.ts            — главный процесс Electron (IPC, tray, hotkeys)
  renderer/
    index.html         — главное окно (React: история, настройки)
    capture.html       — оверлей захвата области + inline редактор
    recording.html     — оверлей записи экрана + превью видео
    editor.html        — отдельный редактор (legacy)
    pages/
      History.tsx      — история файлов
      Settings.tsx     — настройки
public/
  icon.ico             — иконка приложения
```

## Серверное API

### Эндпоинты

| Метод | Путь | Описание |
|---|---|---|
| `POST` | `/api/upload/image` | Загрузка изображения |
| `POST` | `/api/upload/video` | Загрузка видео |
| `GET` | `/api/files/:id` | Получение файла |
| `GET` | `/api/health` | Проверка доступности |

### Формат ответа

```json
{
  "success": true,
  "fileId": "abc123",
  "fileName": "screenshot_2026-03-15_22-51-14.png",
  "url": "http://192.168.1.100:8080/files/abc123",
  "size": 245821
}
```

Авторизация: `Authorization: Bearer <token>` — настраивается в приложении.
