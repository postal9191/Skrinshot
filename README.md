# Skrinshot

Десктопное приложение для Windows — скриншоты, запись экрана, аннотации и загрузка файлов на свой сервер или Яндекс Диск.

---

## Возможности

### Скриншоты
- **Весь экран** — мгновенный захват, открывается встроенный редактор
- **Область экрана** — выделение произвольной области мышью, редактор поверх выделения

### Редактор аннотаций
- Инструменты: стрелка, линия, карандаш, маркер, прямоугольник, эллипс, текст
- Выбор цвета и толщины
- Отмена (Ctrl+Z)
- Действия: **Сохранить** / **Копировать** / **Загрузить** / **Закрыть**

### Запись экрана
- **Весь экран** или **область** — выбирается перед записью
- Индикатор REC внизу, рабочий стол полностью доступен во время записи
- Микрофон и системный звук (loopback)
- После остановки — превью с выбором: сохранить или загрузить
- Формат WebM (VP9), FPS: 15 / 30 / 60, битрейт: 2–20 Mbps

### Загрузка файлов

**Свой сервер**
- HTTP API, `multipart/form-data`, Bearer Token авторизация
- Авто-загрузка после создания, очередь при недоступности сервера

**Яндекс Диск**
- OAuth-авторизация через встроенное окно входа
- Файл автоматически публикуется, ссылка сохраняется в историю
- Папка на диске настраивается (по умолчанию `/Skrinshot`)

### История
- Список скриншотов и видео: дата, тип, размер, статус, ссылка
- Ручная загрузка и удаление записей

### Настройки
- Папка сохранения и шаблон имени файла
- Формат изображений (PNG / JPG), качество JPG
- Параметры видео: FPS, битрейт, микрофон, системный звук
- Загрузка: свой сервер (URL + токен) и/или Яндекс Диск
- Горячие клавиши — кликните на поле и нажмите комбинацию
- Тёмная тема, автозапуск с Windows

## Горячие клавиши по умолчанию

| Действие | Клавиши |
|---|---|
| Скриншот всего экрана | `PrintScreen` |
| Скриншот области | `Ctrl+PrintScreen` |
| Запись экрана | `Shift+PrintScreen` |
| Запись области | `Ctrl+Shift+PrintScreen` |

Все клавиши настраиваются в разделе «Горячие клавиши».

Приложение работает в фоне. Правый клик на иконке в трее:

- Скриншот / Скриншот области
- Запись экрана / Запись области
- История / Настройки / Выйти

Двойной клик — открыть главное окно.

## Установка и запуск

### Требования

- Windows 10 / 11
- Node.js 18+

### Зависимости

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

### Сборка (portable + установщик)

```bash
npm run electron:build
```

Результат в `dist-build/`:
- `win-unpacked/` — portable-версия, запускается без установки
- `Skrinshot Setup 1.0.0.exe` — NSIS-установщик

### Только компиляция (без упаковки)

```bash
npm run build
```

## Свой сервер — API

Поднимается командой `npm run server`. По умолчанию порт `8080`.

### Эндпоинты

| Метод | Путь | Описание |
|---|---|---|
| `POST` | `/api/upload/image` | Загрузка изображения |
| `POST` | `/api/upload/video` | Загрузка видео |
| `GET` | `/api/files/:id` | Получение файла по ID |
| `GET` | `/api/health` | Проверка доступности |

### Формат ответа

```json
{
  "success": true,
  "fileId": "abc123",
  "fileName": "screenshot_2026-03-15_22-51-14.png",
  "url": "http://192.168.1.100:8080/files/abc123",
  "size": 245821
}
```

Авторизация: заголовок `Authorization: Bearer <token>` — токен задаётся в настройках приложения.

## Яндекс Диск — подключение

1. Перейдите на [oauth.yandex.ru/client/new](https://oauth.yandex.ru/client/new)
2. Создайте приложение: название любое, платформа — **Веб-сервисы**
3. Доступы: **Яндекс Диск** (чтение + запись)
4. Redirect URI оставьте по умолчанию: `https://oauth.yandex.ru/verification_code`
5. Скопируйте **Client ID**
6. В настройках Skrinshot → **Сервер** → **Яндекс Диск** → вставьте Client ID → нажмите **Войти через Яндекс**
7. В открывшемся окне войдите в аккаунт → нажмите **Разрешить** → окно закроется автоматически
